# CredCalc — ипотечный калькулятор (Python + Flask)

Простой веб‑калькулятор ипотеки с UI (локально в браузере).

## Возможности
- Ввод: стоимость жилья, первоначальный взнос, срок кредита (лет), процентная ставка (годовых)
- Результат: ежемесячный платеж, переплата (₽) и переплата (%)

## Запуск

1) Установить зависимости:

```bash
pip install -r requirements.txt
```

2) Запустить сервер:

```bash
python app.py
```

3) Открыть в браузере:

`http://127.0.0.1:5000`
```

## Примечания
- Используется аннуитетная схема платежей.
- UI: одна страница с формой ввода и результатом.
- Экспорт графика платежей в Excel: кнопка **«Скачать Excel»** (формат `.xlsx`).
